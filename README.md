## Installation

Install project

```bash
  git clone -b <branch_name> https://github.com/jasontaylor0891/CSC540_project.git
  cd CSC540_project
```
    
## Deployment

To deploy this project run:

```bash
  docker-compose up --build
```

## Local Database Access

```bash
  docker exec -it csc540_project_db_1 mysql -ucsc535 -p
```
